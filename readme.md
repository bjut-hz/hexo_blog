### how to use

- install hexo:

	npm install hexo-cli -g

- clone this rep 

	git clone git@github.com:bjut-hz/hexo_blog.git

- clone submodule
	
	git submodule update --init --recursive


#### hexo usage

- create a new blog

		hexo new post "title"

- start a server at localhost

		hexo server
		hexo s

- deploy to github

		hexo deploy
		hexo d

- clean local database

		hexo clean

- generate html

		hexo generate
		hexo g
